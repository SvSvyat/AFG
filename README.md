# AFG [Anime Funny Game]

AFG is a home project that is being developed TypeScript + Reacr Native.

### How to start

AFG requires [Yarn](https://yarnpkg.com/) to run.

For start developing...

```sh
$ npm install
$ npm install -g expo-cli (if not installed)
$ yarn add expo
$ yarn start
```
